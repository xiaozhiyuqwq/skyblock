# Skyblock-for-Server

### 简体中文

#### 介绍
基于MineCraft基岩版制作的空岛生存存档，用于单机联机与bds服务器。如果您想使用本项目，包括但不限于开服，可以使用项目前身资源（即v1）。

#### 项目
这个项目是源于我策划运营的我的世界基岩版的服务器，您可以访问www.rainyat.work了解这个项目.

#### 语言
目前v1支持简体中文 繁体中文 英语，v2支持简体中文 繁体中文 英语 日语（未适配完成）
