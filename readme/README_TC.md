
# Skyblock-for-Server

### 繁體中文
#### 介紹
基於MineCraft基岩版製作的空島生存存檔，用於單機聯機與bds服務器。如果您想使用本項目，包括但不限於開服，可以使用項目前身資源（即v1）。

#### 項目
這個項目是源於我策劃運營的我的世界基岩版的服務器，您可以訪問www.rainyat.work瞭解這個項目.
